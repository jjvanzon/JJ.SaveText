Before running the Northwind sample, please make sure that:

* You have the Northwind database on your SQL Server

	- If not please see the section "Database Setup" below



Database Setup:
---------------

If you don't have the Northwind database:

(1) Add a database to your SQL Server and give it the name Nortwhind.

(2) Run the script Northwind.sql against your new Northwind databasse to 
create the Northwind schema and fill the database with data.

You can find the file Northwind.sql in the folder /Puzzle/NPersist/Samples/Northwind/Domain
